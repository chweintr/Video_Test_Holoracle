=====================================
TROUBLESHOOTING
=====================================

If test page shows 404:
- Check Railway deployment is complete
- Verify branch is: claude/clone-holoracle-repo-011CUzqwYQ9Y8wchMwYgtTt5

If Simli doesn't load:
- Check SIMLI_API_KEY in Railway variables
- Check Agent ID in config.js
- Check browser console (F12) for errors

If videos don't play:
- Check files exist in assets/videos/
- Check file names match config.js
- Check video format (H.264 or WebM)
